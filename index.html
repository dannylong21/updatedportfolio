About Me

I’m a front-end developer with a sharp eye for design and a deep respect for usability. After two years of production experience and six years as an FDNY EMT, I’ve learned how to solve high-stakes problems with calm precision and human focus. I specialize in HTML, Tailwind CSS, and JavaScript — building responsive, client-ready websites that not only look great, but perform with purpose. My work includes full-stack solutions for real businesses, like a professional landscaping site built with Supabase, dynamic UI components, and real-time features. Every line of code is deployed through GitHub workflows and mobile-tested for users on the go. I approach each project like a product: clean, accessible, fast — and built to convert.

Skills

These are the core technologies I use to build responsive, performant, and client-focused web applications.

JavaScript

I use JavaScript to power dynamic features like scroll animations, modals, and mobile navigation — delivering seamless interactivity that keeps users engaged and clients impressed.

React

I use React to build scalable, component-driven UIs. It’s my go-to for creating reusable layouts and managing state in client projects and personal builds.

Tailwind CSS

Tailwind powers the clean, responsive design of this portfolio. I use it to rapidly prototype and fine-tune layouts that look sharp on every screen — a key factor in attracting clients and recruiters.

Supabase

I use Supabase to handle real-time data, authentication, and backend logic. It powers features like contact forms, admin dashboards, and secure client data workflows in my full-stack builds.

GitHub

I use GitHub to manage version control, collaborate on code, and showcase my work. Every project I build is tracked, documented, and deployed with clean commits and clear structure.
